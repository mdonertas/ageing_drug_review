GenAge Build 19: Release Notes

Release date: June 24, 2017
Number of genes in model organisms: 2,152
Number of human genes: 307

Build 19 is a significant update in terms of content. The set of human ageing genes has again slightly increased, while existing entries have been further curated, resulting in the addition of several bibliographic references. Observations concerning many genes were also updated to reflect new findings.

The model organismsâ€™ set has been significantly extended with nearly 100 new genes associated with longevity added. New references and observations were also added to existing entries.

A few gene symbols were also updated according to the latest nomenclature and the annotation derived from other databases updated.

Please remember these notes refer to the latest stable build of GenAge. The current GenAge stats may differ slightly from the above due to improvements after the build's release.
